Environment : Python + pygame library
Game FPS : 100
Object
	Ball : 16 x 16 px
	Paddle : 100 x 16 px
	Briks : 60 x 16 px

Life system
	Initial Life : 3
	If player miss the ball, life decrease by 1
	If there is any life, it's game over

Level system
	If all bricks clear, level increase
			"			, ball speed increase
			"			, bricks increase

Key
	p : Pause the game
	ESC : End the game
